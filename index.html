About Us

Game Horde is an experimental webapp with the goal of providing a more interactive and helpful experience when discovering, recording, and playing games.

Tools Used

AdminLTE - Open source template for the web design you are seeing right now

MySQL - Database to store game info

SteamAPI - API to get info from steam games

Flask - python REST server for communicating with the webapp and database

Sklearn - python library for basic Machine Learning algorithms
